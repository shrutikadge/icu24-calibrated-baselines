ICU24 Calibrated Baselines

CPU-first, code-only clinical ML baselines for 0–24h ICU outcomes with transparent calibration and subgroup fairness slices.

This site is built from the repository's docs/ folder via GitHub Pages. No real patient data is included; figures below are generated from synthetic data.

Reproduce locally

  1. Create a Python 3.11 environment and install deps:
# macOS example
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-pages.txt  # minimal deps for the synthetic run
  1. Run the synthetic pipeline:
python scripts/generate_synthetic.py
python scripts/run_baselines.py
python scripts/calibrate_and_eval.py
python scripts/make_figures.py
